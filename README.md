# Bank Loan Risk and Performance Analysis

## Project Overview

This project analyzes bank loan data to evaluate loan performance, identify risk patterns, and generate actionable business insights. The analysis is performed using SQL for data processing and Power BI for visualization.

---

## Problem Statement

The objective of this project is to analyze a bank’s loan portfolio to:

* Understand overall loan performance
* Identify good and bad loans
* Evaluate key financial and risk metrics
* Support data-driven decision-making for better loan management

---

## Tools and Technologies

* SQL (MySQL) – Data cleaning and analysis
* Power BI – Dashboard and visualization
* CSV – Data source

---

## Dataset Description

The dataset represents a simulated bank loan portfolio containing:

* Customer details
* Loan attributes
* Financial metrics
* Loan status information

---

## Data Preparation

* Converted date fields into proper date format
* Handled inconsistent date formats
* Validated and structured dataset in SQL
* Ensured correct data types for analysis

---

## Key Metrics

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income Ratio (DTI)
* Good Loan Percentage
* Bad Loan Percentage

---

## Analysis Performed

* Loan performance analysis by status
* Month-to-date (MTD) and previous month comparison
* Regional analysis by state
* Loan term and purpose analysis
* Customer profile analysis (employment length, home ownership)

---

## Dashboard

The Power BI dashboard provides:

* KPI summary for loan performance
* Loan status distribution
* Monthly trends
* Regional insights
* Interactive filtering for detailed analysis

---

## Business Insights

* Majority of loans fall under fully paid and current categories, indicating stable portfolio performance
* Charged-off loans contribute significantly to risk
* Certain customer segments and loan purposes show higher default patterns
* Monthly trends highlight variations in loan applications and repayments

---

## Business Recommendations

* Strengthen credit evaluation for high-risk segments
* Monitor customers with high DTI ratios
* Promote loan products with better repayment performance
* Focus on reducing charged-off loans

---

## Project Structure

* bank_loan_data.sql – SQL queries for data preparation and analysis
* Bank Loan Report.pbix – Power BI dashboard
* problem_statement.sql / document – Business questions and analysis queries
* data_dictionary.md – Explanation of dataset fields and terminology

---

## Conclusion

This project demonstrates how data analysis can be applied to evaluate loan performance, identify risks, and support strategic decision-making in the banking domain.
